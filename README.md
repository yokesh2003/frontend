# Audible Frontend

React + TypeScript frontend for the Audible audiobook platform.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

The app will run on http://localhost:3000

## Features

- User authentication (Login, Register, Forgot Password, Change Password)
- Browse and search audiobooks
- Add to cart and checkout
- Payment processing with card management
- Library management for purchased audiobooks
- Audio player with playback controls
